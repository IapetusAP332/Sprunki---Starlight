# Sprunki---Starlight

Sprunki - Starlight is a fangame of the Incredibox mod Sprunki, created by NyankoBFlol.

The mod includes a seperate set of 20 characters (All made by me) with drastically different instruments and tempo than the original mod, creating a new listening and composing experience.

As with the original mod, it is not suitable for younger users as it contains a more... intense Phase 2. Please review the warning in the mod before continuing.

# Instructions for use

1. Download the .html file.
2. Run it.
3. Enjoy.
